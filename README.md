## Database Chatbot
Chat database using Open-AI Functions using different LLM's.

## Features 
- Query database for input using Open-ai Functions.
- Interact with database using human language and querying it to show result of input.

## Tech
* Flask
* SQL
* Open-AI Functions
* Bootstrap
